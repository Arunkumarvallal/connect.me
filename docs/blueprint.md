# **App Name**: Connect.me

## Core Features:

- Dynamic Grid System: A drag-and-drop grid where users can place 'tiles' of different sizes (1x1, 2x1, 2x2, etc.) to prioritize and rearrange their most important content.
- Tile Library (Widget Types): Enables users to add various content tiles including rich social media embeds (Instagram, X, LinkedIn), high-quality image/video uploads (MP4/GIF), external link buttons with icons and descriptions, live API data integrations (Spotify 'Now Playing,' GitHub commit graphs, YouTube sub counts), and Markdown-supported text/bio blocks.
- Custom Themes: Allows users to personalize their profile's appearance with Light/Dark modes and custom background gradients.
- Custom Profile URLs: Users can claim and customize a unique URL (e.g., mybento.com/username) for their public profile.
- Font Controls: Provides a selection of curated typography options for customizing the profile's text.
- User Account Management: Secure user registration, login, and management of profile settings for their Bento.me page.
- Public Profile Display: A clean, visually-driven public webpage showcasing all the user's curated tiles and branding, accessible via their custom slug.

## Style Guidelines:

- Supports custom background gradients and provides Light/Dark modes for the user's profile.
- Allows users to select from a curated set of typography options for their profile's text.
- Utilize relevant brand logos for social media and custom icons for external links. General interface icons should be clean and minimalist.
- A responsive, modular 'bento-box' grid layout for drag-and-drop arrangement and resizing of tiles, ensuring optimal display on all devices.
